## 自我意识

### 术语

- 反省：对自我的思考
- 自我意识：对自我的定义，也包括了他我，即他人眼中的自己，或者他人对自己的评价
- 自我意识过剩：过于强化他人和自我的联系，或者放大他人眼中的自己
- 自我客体化：过于注重他人对自己的评价

### 链路

- 反省 / 反思 / 自省 / 自审 -> 自我意识
- 自我意识过剩 -> 自我客体化 -> 主体性缺失
- 边界
  - 主观 & 弹性
  - 包含自我意识的最小闭包
