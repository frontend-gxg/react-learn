## 朴素EV

### 公式

- 风险回报比 = risk / reward = 下注 /（底池 + 下注）
- EV(action=下注、跟注、加注，下注) >= EV(action=过牌、弃牌) = 0 <=> 胜率 >= 风险回报比
- 约束：后位跟注、加注的下注尺度 >= 前位下注的下注尺度

### 问题

- 朴素EV是剪枝的结果，忽略了后续的动线

### MDF

- 从对手EV的角度思考，计算对方胜率
- EV = (1 - MDF) * 底池 - MDF * 下注 >= 0 <=> MDF <= 底池 / （底池 + 下注）
